TheAarjav Vulnerable Web Application (TAVWA)

Welcome to TAVWA, an intentionally vulnerable PHP/MySQL web application designed by Aarjav Sharma for security learning and penetration testing practice in a safe environment. This application contains numerous vulnerabilities for educational exploitation.

Practice Areas:
  • SQL Injection (Error & Blind)
  • Command Injection
  • File Upload/Include Exploits
  • Reflected / Stored / DOM-based XSS
  • Session & Access Control Flaws
  • CSRF and SSRF Attacks
  • Weak Cryptography
  • Server Side Template Injection
Disclaimer: Do NOT deploy TAVWA on any live or production server — it is intentionally insecure and meant only for offline labs.
